# Johnson Engineering Design Checklist

## P0 必须通过

- 页面使用 Johnson 工程技术品牌色
- 主色为 `#1E3A8A`
- 页面至少使用一个 Johnson IP 资产
- 页面能看出有限元 / 疲劳 / 振动 / 耐久 / 代码 / AI 的工程属性
- 页面不是通用 AI 模板风
- 删除旧品牌名、旧头像和旧人物描述痕迹
- 图片路径全部为本地 assets 相对路径
- 响应式布局正常
- HTML 无默认浏览器样式感

## P1 应该通过

- 有知识卡片模块
- 有工程图标或贴纸
- 有清晰的信息层级
- 有明确结论或 CTA
- 图文比例合理
- 字号清晰
- 卡片留白充足

## P2 加分项

- 使用 `assets/banner-engineering.png`
- 使用 emoji 表情增强互动
- 使用 sticker 贴纸增强工程识别
- 使用线稿图作为低饱和装饰
- 适合截图传播
- 适合 PPT 或技术文档复用
