# Scene: App / 工具界面 / 工程仪表盘

适用于仿真流程工具、代码工具、数据分析看板、工程可视化和 AI 辅助分析界面。

## 内容特征

- 工具界面
- 仿真流程
- 代码工具
- 数据分析
- 仪表盘
- 工程可视化

## 推荐结构

1. Header：工具名称、Johnson 头像、主要状态
2. Sidebar / Tabs：FEA、Fatigue、Vibration、Durability、Code、AI
3. Main Panel：输入、流程、图表、结论
4. Result Cards：通过、风险、建议、下一步
5. Export / CTA：导出报告、复制代码、生成说明

## 视觉规则

- App 页面比 Landing 更克制，减少大装饰
- 主交互色为工程蓝 `#1E3A8A`
- 成功、通过、AI 辅助结果用科技绿 `#10B981`
- 风险、警告、疲劳损伤用活力橙 `#F97316`
- 卡片白底、浅灰边框、明确 hover/focus 状态
