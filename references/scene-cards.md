# Scene: 社交知识卡片 / 工程图文

适用于短内容、工程知识拆解、分析结论、教程摘要和适合截图传播的图文卡片。

## 内容特征

- 社交知识卡片
- 短内容
- 亮色
- 高识别度
- 工程图标
- 适合截图传播

## 卡片规格

- 推荐尺寸：1080 x 1440px
- 页面风格：明亮黄知识卡片背景 + 工程蓝标题
- 作者信息：Johnson / Simulation Engineer
- 每张卡片只讲一个工程观点

## 推荐结构

1. Cover：标题、Johnson 头像、工程主题贴纸
2. Problem：工程问题和风险
3. Method：分析路径、公式、仿真流程
4. Result：结果、曲线、云图、判断
5. Summary：一句话结论 + 关注/联系 CTA

## 视觉规则

- 避免信息过密，每页留出呼吸感
- 标题用工程蓝，重点块用明亮黄
- 疲劳和风险用活力橙，AI 和通过结论用科技绿
- 可使用 `assets/emoji/*` 增强互动，但不能盖过工程结论
- 不使用旧品牌署名、旧头像或旧人物描述

## 必检项

- 头像路径为 `assets/avatar.jpg`
- 至少一张贴纸来自 `assets/stickers/`
- 最后一页包含 Johnson / Simulation Engineer
- 所有图片路径为相对路径
