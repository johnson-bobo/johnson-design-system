# Scene: Landing / 个人主页 / 能力展示

适用于 Johnson 工程能力展示、个人主页、服务介绍、课程或工具发布页。

## 内容特征

- 工程能力展示
- 仿真能力
- 技术路线
- 个人品牌
- IP Banner
- 核心服务

## 推荐结构

1. Hero：使用 `assets/banner-engineering.png` 或 `assets/character.png`
2. Capability：FEA / Fatigue / Vibration / Durability / Code / AI 六类能力
3. Workflow：从问题定义到仿真、校核、结论和工具化
4. Proof：项目经验、方法论、交付物示例
5. CTA：联系、订阅、查看案例或启动工程分析

## 视觉规则

- Hero 用工程蓝标题和明亮黄背景氛围
- 能力卡片使用贴纸图标增强识别度
- CTA 按钮使用工程蓝，次按钮使用明亮黄
- 页面应该专业、亲和、清晰，不做蓝紫赛博朋克
