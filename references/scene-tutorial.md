# Scene: 工程教程 / 技术说明

适用于有限元、疲劳、振动、耐久、代码工具和 AI 辅助工程分析教程。

## 内容特征

- 工程教程
- 步骤清晰
- 公式 / 图表 / 结论
- 有限元 / 疲劳 / 振动 / 耐久
- 知识卡片

## 推荐结构

1. Hero：主题、工程问题、Johnson IP 头像或半身人物
2. Problem：工程问题、约束、输入条件
3. Method：分析方法、公式、仿真流程
4. Result：云图、曲线、频谱、寿命结论
5. Checklist：风险点、校核项、下一步

## 视觉规则

- 背景使用暖白 `#FEFCF6`
- 标题、步骤编号和关键结论用工程蓝 `#1E3A8A`
- 通过 / 正向 / AI 辅助用科技绿 `#10B981`
- 风险 / 疲劳 / 警示用活力橙 `#F97316`
- 知识卡片、重点块用明亮黄 `#FACC15`
- 优先使用 `assets/character-half.png`、`assets/emoji/emoji-think.png`、`assets/emoji/emoji-done.png`
